# 🍅 또마니또 (To-Manito) — Self Demo 가이드
 
직접 로컬 환경에서 또마니또를 실행하고 핵심 기능을 체험할 수 있는 시연 가이드입니다.
 
---

## 🌐 배포 버전으로 바로 체험하기
 
로컬 환경 설정 없이 바로 체험하려면 아래 링크를 이용하세요.
데모 시나리오는 로컬 실행 방법 하단에서 확인할 수 있습니다.
 
👉 [https://tomanito.vercel.app/](https://tomanito.vercel.app/)

---
 
## 🛠 로컬 실행 시 사전 준비
 
| 항목 | 버전 |
|------|------|
| Flutter | 3.41.5 이상 |
| Dart | 3.11.3 이상 |
| Java | 17 |
| Gradle | 프로젝트 내 wrapper 사용 |
 
환경 변수 설정이 필요합니다. 프로젝트 루트의 `.env` 또는 `application.yml`에 아래 값을 채워주세요.
 
```
DB_URL=***
DB_USER=***
DB_PASSWORD=***
JWT_SECRET=***
GEMINI_API_KEY=***
```

## 🚀 실행 방법
 
### 1️⃣ 레포지토리 클론
 
```bash
git clone https://github.com/team-capstone-reverdir-2026/reverdir_repo.git
cd reverdir_repo
```
 
### 2️⃣ 백엔드 실행
 
```bash
cd backend
./gradlew build
./gradlew bootRun
```
 
> 백엔드가 정상 실행되면 `http://localhost:8080` 에서 API가 응답합니다.
 
### 3️⃣ 프론트엔드 실행
 
```bash
cd frontend
flutter pub get
flutter run -d chrome
```
 
> 브라우저에서 또마니또 앱이 열립니다.

## 🎮 데모 시나리오
 
아래 순서대로 따라하면 또마니또의 핵심 기능을 한 사이클로 체험할 수 있습니다.
 
> 💡 **권장 인원**: 최소 2명 (각자 다른 브라우저 탭 또는 기기로 접속)  
> 혼자 체험하는 경우 다른 종류의 브라우저를 활용해 두 계정으로 진행하세요.
 
---
 
### STEP 1 — 회원가입 & 로그인
 
1. 앱 접속 후 **회원가입** 진행
2. 두 번째 사용자도 동일하게 회원가입 (다른 브라우저 활용)
3. 각각 로그인
---
 
### STEP 2 — 방 생성 & 입장
 
**방장 계정 (사용자 A)**
1. 홈 화면에서 **방 만들기** 선택
2. 방 이름, 기간 설정 후 생성
3. 생성된 **방 코드** 또는 링크 복사
**참여자 계정 (사용자 B)**
1. 홈 화면에서 **방 참여하기** 선택
2. 방 코드 입력 후 입장
---
 
### STEP 3 — 미션 입력
 
**각 참여자**
1. 게임 시작 전 **미션 입력** 화면에서 미션 3개 작성
2. 직접 입력하거나 **추천 미션** 버튼으로 랜덤 미션 사용 가능
3. 입력 완료 후 **입장 완료** 버튼 클릭
---
 
### STEP 4 — 게임 시작 & 마니또 매칭
 
**방장 계정 (사용자 A)**
1. 모든 참여자 입장 확인 후 **게임 시작** 버튼 클릭
2. 서버에서 자동으로 마니또 매칭 처리
3. 각 참여자는 자신의 **마니띠(내가 챙겨야 할 상대)** 확인 가능
---
 
### STEP 5 — 오늘의 질문 확인 & 답변
 
1. 게임 화면에서 **오늘의 질문** 탭 이동
2. 오늘 제공된 질문 확인 (방마다 다른 질문 제공)
3. 본인 답변 작성 후 제출
4. **마니또(나를 챙기는 상대)의 답변**을 보며 그 사람이 누구인지 추리
> 💡 질문 예시: "무인도에 하나만 가져갈 수 있다면?", "요즘 가장 좋아하는 음식은?"
 
---
 
### STEP 6 — 익명 쪽지 전송
 
1. **익명 쪽지** 탭 이동
2. 마니띠에게 익명으로 쪽지 작성 & 전송
3. **받은 쪽지함**에서 수신 쪽지 확인
4. 발신자 정보는 게임 종료 전까지 비공개
---
 
### STEP 7 — 게임 종료 & 결과 공개
 
**방장 계정 (사용자 A)**
1. **게임 종료** 버튼 클릭
2. **결과 보기** 클릭 시 전체 마니또-마니띠 매칭 결과 확인 가능
3. '너였어?', '너 이런 노래 자주 들어?' 등 게임 진행 중 쌓인 정보로 대화 가능
---
 
### STEP 8 — AI 결과 리포트 확인
 
1. 결과 확인 후 **내 결과 분석 리포트 보러 가기** 버튼으로 이동
2. Gemini 기반 개인 맞춤형 행동 분석 리포트 확인
   - 쪽지 전송 횟수, 평균 메시지 길이, 주요 활동 시간대 등 정보를 활용해 참여 스타일 서술
   - 나의 참여 스타일을 한 줄 캐릭터로 표현
3. 다양하게 개인화된 리포트 결과를 대화 주제로 활용 가능

## ✅ 핵심 체크포인트
 
| 기능 | 확인 포인트 |
|------|------------|
| 마니또 매칭 | 자기 자신에게 매칭되지 않는지 확인 |
| 오늘의 질문 | 같은 방 참여자 모두에게 동일한 질문이 제공되는지 확인 |
| 익명 쪽지 | 발신자 정보가 종료 전까지 숨겨지는지 확인 |
| AI 리포트 | 게임 종료 후 개인별 리포트가 생성되는지 확인 |
