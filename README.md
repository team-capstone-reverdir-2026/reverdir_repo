## 🚀 Overview  
<br>

>유저 간 펜팔 형태의 소통을 통해
>책을 추천받고, 자신의 독서 경험을 기록할 수 있는 서비스입니다.
><br><br>
>기존 독서 앱의 단조로운 기록 방식에서 벗어나,  
>소셜 요소 + 게임 요소를 결합하여 독서를 더 재미있고 지속 가능한 활동으로 만드는 것을 목표로 합니다.

<br>
