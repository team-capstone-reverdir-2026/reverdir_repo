
# 🎅 ProjectName

<br><br>

---

## 🚀 Overview  
<br>

>지인 기반 그룹에서 진행되는 마니또 활동을
>익명성, 실시간 상호작용, 게임화 요소를 통해 확장한 모바일 앱 서비스입니다.
><br><br>
>단순한 매칭과 결과 발표를 넘어,
>익명 채팅, 미션 수행, 힌트 제공, 의심 투표, 활동 기반 랭킹 등을 포함하여
>마니또를 참여형 게임 경험으로 재구성하는 것을 목표로 합니다.

<br>

---

## 🎯 Target & Problem
### 👤 Target User
> - 대학 동아리 및 학과 단체
> - 회사 팀 및 조직 내 소규모 그룹
> - 친구 모임 및 사적 커뮤니티

### ⚠️ Pain Point

> - 종이 추첨, 수동 관리 등 수작업 중심의 비효율적인 진행 방식
> - 단순 선물, 배려 구조로 인한 낮은 참여도와 빠른 흥미 감소
> - 개인 간 관계 중심 진행으로 그룹 전체 상호작용 부족
> - 단순한 구조로 인한 약한 익명성과 게임적 긴장감 부재
