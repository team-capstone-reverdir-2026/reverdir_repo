
# 🍅 또마니또
<br>

> 익명 매칭부터 맞춤형 미션, 쪽지, 힌트, 랭킹, AI 기반 활동 리포트까지 마니또 활동 전 과정을 지원해 지인들 간 소모임의 마니또 이벤트 질을 높이는 소셜 앱
<br><br>

---

## 🚀 Overview  
<br>

>지인 기반 그룹에서 진행되는 마니또 활동을
>익명성, 실시간 상호작용, AI 분석, 게임화 요소를 통해 확장한 모바일 앱 서비스입니다.
><br><br>
>단순한 매칭과 결과 발표를 넘어,
>방의 성격에 맞춘 AI 미션 생성, 실시간 채팅, 실시간 랭킹 보드, 그리고 마니또 활동 종료 후 제공되는 'AI 마니또 캐릭터 리포트'를 통해
>참여자들에게 깊은 재미와 새로운 상호작용 경험을 제공합니다.

<br>

---

## 🎯 Target & Problem
### 👤 Target User
> - 대학교 학과/동아리 동기 모임
> - 회사 팀 및 사내 소규모 조직
> - 온라인 게임 길드 등 공통의 관심사를 가진 커뮤니티

### ⚠️ Pain Point

> - 종이 추첨, 수동 관리 등 수작업 중심의 비효율적인 진행 방식
> - 단순 선물, 배려 구조로 인한 낮은 참여도와 빠른 흥미 감소
> - 마니또-마니띠 간의 1:1 관계에만 치중되어 그룹 전체의 상호작용 부족
> - 참여자의 활동 내역이 휘발되어 종료 시점의 결과 발표가 단조로움
<br>

---

## ✨ Key Features
<br>

> - 🎲 초대 코드 기반 마니또 방 생성 및 익명 자동 매칭
> - 🖥 AI 기반 맞춤형 환경 구성
>   - 방 생성 시 입력된 메타데이터(모임 성격)를 바탕으로 AI가 그룹 특성에 맞는 추천 미션과 힌트 질문을 자동 생성
> - 📋 게임화된 활동 시스템
>   - 나의 마니또로 의심되는 상대 설정
>   - 마니또 힌트 전송
>   - 내 마니띠에게 비밀 쪽지 전송
>   - 내가 받은 / 준 선물 기록
> - 🔍 힌트 전송 및 의심 마니또 투표
> - 🏆 활동량 기반 실시간 랭킹 및 정체 공개 정산
> - 🤖 게임 종료 후 AI 기반 개인 활동 리포트 & 나의 캐릭터 유형 분석
